## 🔄 The Complete Flow

```
START
  │
  ├─► Load IXL_ALGEBRA1_LINKS (297 URLs)
  │
  ├─► For each URL:
  │     │
  │     ├─► Query Supabase: Does URL exist in database?
  │     │     │
  │     │     ├─► YES → Skip (already processed)
  │     │     │
  │     │     └─► NO → Continue to processing:
  │     │           │
  │     │           ├─► 1. Scrape with Firecrawl
  │     │           │      - Capture screenshot
  │     │           │      - Extract page content
  │     │           │
  │     │           ├─► 2. Analyze with Gemini AI
  │     │           │      - Extract LaTeX question
  │     │           │      - Identify visual elements
  │     │           │      - Classify question type
  │     │           │
  │     │           └─► 3. Save to Supabase
  │     │                  - subject: "Algebra 1"
  │     │                  - subtopic: from mapping
  │     │                  - question_latex: extracted
  │     │                  - question_type: classified
  │     │                  - website_link: original URL
  │     │                  - visual_elements_url: screenshot
  │     │
  │     └─► Next URL
  │
END
```
